# 双足机器人

## 1.环境配置
- 1.1 添加虚拟环境 + 激活虚拟环境
    ```  
    virtualenv micro-507 --python=python3
    cd micro-507
    source Scripts/activate
    cd ..
    ```
- 1.2 安装依赖
    ```
    pip install pybullet
    pip install qpsolvers  
    pip install osqp  
    python qp.py  测试安装
    出现 QP solution: x = [ 0.3076548  -0.69232615  1.38448776] 则安装成功
    ```
- 1.3 可以使用如下命令退出虚拟环境   
  ```
  deactivate
  ```
